## Overview

MyLang Reader's vocabulary system automatically tracks every word you encounter, building a personalized learning database. This guide shows you how to make the most of vocabulary management features.

## How Vocabulary Tracking Works

### Automatic Collection

Every word you click in the reader is automatically:
- **Added to your vocabulary database**
- **Categorized by familiarity level**
- **Tagged with context** from where you encountered it
- **Tracked for frequency** and learning progress

### Familiarity Levels

<Tabs>
  <Tab title="Learning (Yellow)">
    Words you're actively studying:
    - New words you want to learn
    - Words you partially understand
    - Vocabulary requiring practice
    - Appear in flashcard sessions
  </Tab>
  <Tab title="Known (Green)">
    Words you know well:
    - Comfortable using in context
    - No longer need translation
    - Excluded from practice by default
    - Help track your progress
  </Tab>
  <Tab title="Difficult (Red)">
    Challenging words that need extra attention:
    - Complex or abstract concepts
    - Words with multiple meanings
    - Require more repetition
    - Prioritized in practice sessions
  </Tab>
  <Tab title="New (Blue)">
    Recently encountered, uncategorized words:
    - First time clicking
    - Awaiting your categorization
    - Automatically suggested for review
    - Convert to other categories as you learn
  </Tab>
</Tabs>

### Optimal Vocabulary Load

<AccordionGroup>
  <Accordion title="Beginners (A1-A2)">

    - **Strategy**: Master basics before advancing

  <Accordion title="Intermediate (B1-B2)">

  </Accordion>

  <Accordion title="Advanced (C1-C2)">

    - **New words per day**: 15-25
    - **Focus**: Nuanced meanings, idiomatic expressions
    - **Strategy**: Deep understanding and usage
  </Accordion>
</AccordionGroup>

## Integration with Practice System

### Automatic Practice Generation

Your vocabulary lists automatically feed into the practice system:

- **Learning words** appear in flashcard sessions
- **Difficult words** get extra repetition
- **Known words** are excluded unless requested
- **New words** are suggested for learning sessions

### Spaced Repetition Algorithm

The system schedules reviews based on:
- **Performance history** - harder words appear more frequently
- **Time since last review** - longer intervals for well-known words
- **Learning goals** - customize review frequency
- **Available time** - adapt to your schedule

## Troubleshooting Common Issues

<AccordionGroup>
  <Accordion title="Duplicate words appearing">
    - System may track different word forms separately
    - Use merge function to combine related entries
    - Check for spelling variations or typos
    - Contact support for persistent issues
  </Accordion>

  <Accordion title="Words not saving correctly">
    - Ensure stable internet connection
    - Check if logged into account
    - Try refreshing the page
    - Verify browser allows local storage
  </Accordion>

  <Accordion title="Categories not updating">
    - Changes may take a moment to sync
    - Refresh the vocabulary page
    - Check if in guest mode vs. account mode
    - Clear browser cache if needed
  </Accordion>

  <Accordion title="Missing vocabulary data">
    - Check if logged into correct account
    - Verify sync is enabled
    - Look for backup/export files
    - Contact support for data recovery
  </Accordion>
</AccordionGroup>

## Best Practices Summary

### Daily Workflow

1. **Review new words** from recent reading sessions
2. **Categorize uncategorized words** (5-10 minutes)
3. **Practice flashcards** with learning vocabulary
4. **Update familiarity levels** based on practice performance

### Weekly Maintenance

1. **Clean up vocabulary lists** - remove unwanted words
2. **Export progress** for backup
3. **Analyze learning statistics** for insights
4. **Adjust learning goals** based on progress

### Monthly Review

1. **Assess vocabulary growth** trends
2. **Identify learning patterns** and gaps
3. **Update learning strategies** based on performance
4. **Set new vocabulary goals** for the coming month
