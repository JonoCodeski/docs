## Overview

The MyLang Reader News Feed provides access to articles from international news sources in multiple languages. This feature helps you discover authentic, current content for language learning while staying informed about global events.

<Info>
The News Feed is currently in active development. Some sources may not work perfectly, but we're continuously improving coverage and functionality based on user feedback.
</Info>

## Troubleshooting

### Common Issues

<AccordionGroup>
  <Accordion title="Articles not loading">
    **Possible causes**:
    - Slow internet connection
    - Source website temporarily unavailable
    - Content blocked by ad blockers or browser extensions

    **Solutions**:
    - Refresh the news feed
    - Try accessing individual articles directly
    - Disable browser extensions temporarily
    - Check internet connection stability
  </Accordion>

  <Accordion title="Import failures">
    **Common problems**:
    - Article behind paywall or subscription
    - Complex page layout preventing extraction
    - Content protection preventing copying

    **Alternatives**:
    - Try different articles from same source
    - Copy and paste text manually if available
    - Use alternative sources for similar content
    - Report problematic sources for improvement
  </Accordion>

  <Accordion title="Language detection errors">
    **When it happens**:
    - Mixed language content
    - Short article excerpts
    - Technical or specialized content

    **How to fix**:
    - Manually set language after import
    - Use longer articles for better detection
    - Report persistent issues to support
  </Accordion>

  <Accordion title="Content not updating">
    **Troubleshooting steps**:
    - Force refresh with Ctrl+F5 (Windows) or Cmd+R (Mac)
    - Clear browser cache for the site
    - Check if other users report similar issues
    - Contact support if problem persists
  </Accordion>
</AccordionGroup>
